# HGN v188 — Ad Manager

Adds:

- `/admin/ads`
- `/admin/ads/[id]`
- `/admin/ads/placements`
- `ads` table
- `ad_placements` table
- reusable `AdSlot` component

## What you can do

- create ads
- edit ads
- assign placement
- add image URL
- add destination URL
- add alt text
- add optional HTML code
- set start/end dates
- activate/pause/draft/delete

## SQL to run

`supabase/v188-ad-manager.sql`
