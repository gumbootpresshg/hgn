# HGN v61 Beta Ready Upgrade

This package is an upgrade patch for the HGN Next.js/Supabase site.

## Install
1. Unzip over `C:\HGN\HGNSite`.
2. Run SQL in `supabase/v61-upgrade.sql`.
3. Run:

```powershell
cd C:\HGN\HGNSite
hgnreset
```
